Photo Caption Cleaner

Attempts to cleanup photo Caption, Credit, Description and Alt text in Wordpress

With a Media Library Modal open (the pop-up window that appears when you click to set a Featured Image or Add Media) and an image selected, or while viewing an image in the Media Library, clicking this bookmarklet.

It will copy the raw Caption to the Description field, attempt to extract the Photo Credit and set it in the correct field, and attempt to trim the Getty Dateline and Credit from the Caption field. It will also set basic Alt Text for you, and remove "APTOPIX" from the Title, if it's there.

Note: Some instances of month names may be abbreviated when they should not be.

This feature is still considered experimental.
